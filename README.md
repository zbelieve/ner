这是一个利用传统的bootstrapping方法利用模板进行判断的关系抽取项目
这个前面走了一些弯路，由于是传统方法所以代码比较多，第一次做这个，做的有点乱

2.data是数据以及数据预处理

可以自己搜索jieba切分特殊字符
